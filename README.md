## vscode常用快捷键
- 新建文件 Ctrl+N
- 文件之间切换 Ctrl+Tab
- 代码格式化 Shift+Alt+F
- 上下移动一行 Alt+Up 或 Alt+Down
- 上下复制一行 Shift+Alt+Up 或 Shift+Alt+Down
- 在当前行插入一行 Ctrl+Enter
- 在当前行上方插入一行 Shift+Ctrl+Enter
